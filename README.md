## 🛠 How do I install and run this? (Step-by-Step for absolute beginners!)

We made this super easy. Just follow these steps, even if you don't know anything about coding!

### Step 1: Download the App
Click the **green "<> Code"** button at the top right of this page and click **Download ZIP**. 
Extract the folder to your desktop.

### Step 2: Install Node.js
If you don't have Node.js installed, go to [NodeJS.org](https://nodejs.org/) and download the recommended version. Install it like any normal program.

### Step 3: Get your free Google Gemini API Key
This app uses Google's AI brain (Gemini).
1. Go to [Google AI Studio](https://aistudio.google.com/).
2. Sign in with your Google account.
3. Click on **"Get API Key"** and create one. Copy this long text.

### Step 4: Add your Key to the App
1. Open the extracted folder.
2. Find the file called `.env.example` and rename it to exactly `.env` (just remove the `.example` part).
3. Open this `.env` file using Notepad (or any text editor) and replace `your_gemini_api_key_here` with the long key you copied from Google. Save the file.

### Step 5: Start the App!
1. Open your computer's **Command Prompt** (on Windows: search "cmd" in the start menu) or **Terminal** (on Mac).
2. "cd" into the folder where you placed the app. For example: `cd Desktop/Automated-Reels-Creation-for-YouTube-Automation-`
3. Type: `npm install` and hit Enter. Wait for it to finish.
4. Type: `npm run dev` and hit Enter!
5. Open your web browser and go to the link it shows (usually `http://localhost:3000` or `http://localhost:5173`).

**🎉 BOOM! You're inside the Matrix. Start generating!**
